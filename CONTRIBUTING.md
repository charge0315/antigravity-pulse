## 🌟 How Can I Contribute?

### Reporting Bugs
- Check the [Issues](https://github.com/charge0315/antigravity-pulse/issues) to see if the bug has already been reported.
- If not, create a new issue. Include a clear title, a description of the problem, steps to reproduce, and any relevant logs or screenshots.

### Suggesting Enhancements
- Enhancement suggestions are tracked as [GitHub issues](https://github.com/charge0315/antigravity-pulse/issues).
- Describe the feature you'd like to see, why it would be useful, and how it might work.

### Pull Requests
1. **Fork the repo** and create your branch from `main`.
2. **Implement your changes**. Ensure your code follows the project's style and is well-documented.
3. **Run tests** to make sure everything is still working as expected.
4. **Submit a Pull Request** with a clear description of your changes.

## 💻 Development Environment

To set up your environment, follow the instructions in the [README.md](./README.md#🚀-getting-started).

### Tech Stack
- **Backend**: Rust (Tauri v2)
- **Frontend**: React (Tailwind CSS)
- **API**: Windows Audio Session API (WASAPI)

### Coding Standards
- **Rust**: Follow the standard Rust idioms. Use `cargo fmt` and `cargo clippy`.
- **TypeScript**: Use functional components in React. Keep CSS organized with Tailwind.
- **Documentation**: Provide comments for complex logic, especially when dealing with Win32/COM.

### 🐛 バグ報告
- すでに同じ問題が [Issues](https://github.com/charge0315/antigravity-pulse/issues) に報告されていないか確認してください。
- 新しく報告する場合は、再現手順や OS のバージョン、ログなどを添えて詳細に記述してください。

### ✨ 機能提案
- 新機能のアイデアは [GitHub issues](https://github.com/charge0315/antigravity-pulse/issues) で受け付けています。
- なぜその機能が必要か、どのように動作するかのイメージを共有してください。

### 🚀 プルリクエスト (PR)
1. リポジトリを **Fork** し、`main` からブランチを作成してください。
2. コードを実装し、テストが通ることを確認してください。
3. 変更内容を分かりやすく記載して **Pull Request** を送信してください。
